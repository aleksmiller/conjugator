# Conjugator

Master verb conjugations with ease - the smart way to learn and practice verb conjugations in any language.

## GitHub Pages

This repository includes a GitHub Pages site located in the `docs/` folder with the following pages:

- **Home** (`index.html`) - Promo page featuring the app's benefits and download links
- **Privacy Policy** (`privacy.html`) - Comprehensive privacy policy
- **Support** (`support.html`) - FAQ, contact form, and support resources

To enable GitHub Pages for this repository:
1. Go to repository Settings
2. Navigate to Pages section
3. Select "Deploy from a branch"
4. Choose `main` branch and `/docs` folder
5. Click Save

The site will be available at: `https://<username>.github.io/conjugator/`
